### 📚 What I Learned Today

* **Infrastructure as Code (IaC):** Managing infrastructure using code instead of manual configurations—making deployments faster, consistent, and less error-prone.
* **What is Terraform?** A leading cloud-agnostic IaC tool that helps manage infrastructure across multiple providers using a unified workflow.
* **How Terraform Works:**

  * `init` → Initialize the working directory
  * `plan` → Preview changes (dry run)
  * `apply` → Execute and provision resources

### ⚙️ Setting Up My Environment

* Installed Terraform using official documentation
* Created an alias for efficiency: `alias tf='terraform'`
* Enabled auto-completion with `terraform -install-autocomplete`
* Integrated Terraform with VS Code for better readability and productivity
